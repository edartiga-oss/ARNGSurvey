☁ Google Drive setup

To save reports to your Google Drive, this app needs a free Google OAuth Client ID. This is a one-time setup and the ID is stored only in this browser.

  1. Open the Google Cloud Console and create (or pick) a project.
  2. In APIs & Services → Library, enable the Google Drive API.
  3. In APIs & Services → OAuth consent screen, choose External, fill in an app name + your email, and add your Google account under Test users.
  4. In APIs & Services → Credentials, create an OAuth client ID of type Web application.
  5. Under Authorized JavaScript origins, add this site's origin:
  6. Copy the Client ID and paste it below.

Uses the drive.file scope, so the app can only see the files it creates here — never the rest of your Drive.
